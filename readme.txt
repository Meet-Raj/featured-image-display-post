== Description ==

This plugin adds a "Thumbnail" column to the post list in the WordPress backend, allowing you to see the featured image of each post at a glance. The featured image, also known as the post thumbnail, is a crucial visual element that can enhance the presentation of your posts, especially in themes that utilize featured images prominently.

== Short Description ==

Adds a "Thumbnail" column to the post list in the WordPress backend, displaying featured images for each post.

== Installation ==

1. Upload the `featured-image-display-post` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, navigate to the "Posts" screen in the WordPress admin area.
4. You will now see a new column labeled "Thumbnail" in the post list table.
5. If a post has a featured image set, the thumbnail of the featured image will be displayed in this column. If not, it will show "No Thumbnail".
